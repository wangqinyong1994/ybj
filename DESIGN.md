# 居家服务网站 - 页面设计稿

## 设计规范

### 色彩体系
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | rgb(28,118,44) | 按钮、重点文字、图标 |
| 主色浅 | rgb(56,176,82) | hover状态、次要按钮 |
| 主色深 | rgb(20,85,32) | 按下状态、阴影 |
| 辅助暖色 | rgb(255,152,0) | 温馨感点缀、促销标签 |
| 背景灰 | rgb(248,249,250) | 页面背景 |
| 文字主色 | rgb(33,37,41) | 正文文字 |
| 文字辅色 | rgb(108,117,125) | 次要文字 |

### 响应式断点
- PC端：≥1200px
- 平板：768px - 1199px
- 手机：<768px
